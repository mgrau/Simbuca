1) make all particles in separate file

2) create png images of each time

see draw_movie.C, to compile do

g++ `root-config --cflags` -c draw_movie.C
g++ -o draw_movie draw_movie.o `root-config --libs`

3) Run draw_movie as, e.g

$./draw_movie 

in a folder in which you have all the Simbuca output txt files for all particles (and no other file!)

This step will take some time, so do it in the backgroud and let it run overnight

4) make a list of output files in a separate file:

$ for x in `ls -tr *.png`; do echo $x > listfiles.txt; done

5) Create movie from the files listed in listfiles.txt

mencoder mf://@listfiles.txt -mf fps=25 -o movie.avi -ovc lavc -lavcopts vcodec=msmpeg4v2:vbitrate=1600
